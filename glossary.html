Basics

To be able to edit code the section should be unlocked, however for it to be applicable in your project it needs to be locked. This of it as an "Edit mode."

This allows you to hide of show the selected Menu, it can make it easier to keep track of what you're working on.

The Editor and the Initial State must both be filled before being able to execute your project.

When selecting two or more elements remember you can only perform one action at a time. So, if you want to delete two elements you are going to have to do so individually.

This is especially significant when trying to execute your program.

Editor

The editor is where you build what exists or could happen in your project.



Initial State

The Initial State is the base in which your program will start or build from.
